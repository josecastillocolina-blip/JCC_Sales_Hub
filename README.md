# JCC Sales Hub 
 
### Sistema de Automatización de Fuerza de Ventas (SFA) 
 
Solución tecnológica diseñada para la optimización de procesos de ventas, gestión de inventarios y sincronización de datos en tiempo real. 
 
#### Tecnologías principales: 
* **Backend:** Python / Django 
* **Entorno:** Docker 
* **Control de Versiones:** Git / GitHub
